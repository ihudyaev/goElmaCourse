Дан массив A, состоящий из N целых чисел. 
Вращение массива - это сдвиг каждого элемента вправо на один индекс, а последний элемент массива перемещается на первое место. Например, поворот массива A = [3, 8, 9, 7, 6] равен [6, 3, 8, 9, 7] (элементы сдвигаются вправо на один индекс, а 6 перемещается на первое место).
Задача сдвинуть массив А, N раз, то есть, каждый элемент будет сдвинут вправо N раз
Пример
A = [3, 8, 9, 7, 6] K = 2
Результат: [7, 6, 3, 8, 9]
